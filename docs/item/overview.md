# アイテム開発

XRift アイテムの開発に関するドキュメントです。

## 概要

XRift アイテムは、ワールド内に配置して使える再利用可能な 3D コンポーネントです。React Three Fiber で実装し、Module Federation で配布されます。アップロードしたアイテムはインベントリから各ワールドで使用できます。

## 開発の流れ

1. `xrift create item` でアイテムプロジェクトを作成
2. `src/Item.tsx` を編集してアイテムを実装（`npm run dev` でローカルプレビュー）
3. `npm run build` でプロダクションビルド
4. `xrift check item` でセキュリティチェック（任意）
5. `xrift upload item` で XRift プラットフォームにアップロード
6. 自動審査を経てアイテムが公開され、インベントリから使用可能になる

## ワールドとの違い

| 項目 | アイテム | ワールド |
|------|---------|---------|
| 用途 | ワールドに配置する再利用可能なコンポーネント | インタラクティブな 3D 空間全体 |
| エクスポート形式 | `Item` コンポーネントと `ItemProps` | `World` コンポーネント |
| `xrift.json` の設定 | `item` キー配下 | `world` キー配下 |
| 物理 / カメラ / 出力バッファ設定 | 持たない | ワールド側で設定可能 |
| 利用場所 | 各ワールドのインベントリから配置 | プラットフォーム上のインスタンスとして公開 |
